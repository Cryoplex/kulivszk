Como entrar al servidor de Minecraft

Instalación de FTB

  • Descargar FTB.jar
  • Meter FTB.jar en una carpeta (Por ejemplo, en una carpeta dentro del escritorio)

    Nota para usuarios de Linux: Marcar archivo como ejecutable en Propiedades

  • Hacer click derecho en FTB.jar y seleccionar Abrir con
  • Seleccionar Java en la lista y abrir el archivo.

Configuración

  • Aparecerá un mensaje diciendo Launcher Update Available
  • Presionar el botón No
  • Vamos al seleccionador donde pone Select profile y lo abrimos.
  • Seleccionamos Create Profile, escribimos el nick que queramos y la contraseña que queramos.
  • Presionamos Add
  • Nota: Aunque aparezca la opción "Remember password", debido a un bug cada vez que abramos el lanzador FTB.jar se reseteará el perfil y habrá que volver a crearlo.
  • Seleccionamos la pestaña MODPACKS y buscamos FTB Lite 2
  • Pulsamos el botón Launch Si aparece este mensaje, pulsa Aceptar.

Instalacion

  • Cuando haya terminado de descargarse el pack y se haya abierto la ventana donde pone MOJANG, cerramos Minecraft.
  • Vamos a la pestaña Options y copiamos la ruta que aparece al lado de Install folder..., por ejemplo Z:/home/kulivszk/Escritorio/Minecraft/FTB
  • Abrimos la carpeta que aparece en esa ruta, pero todavía no vamos a hacer nada.
  • Descargamos esto: Pack de Mods
  • Vamos a la ruta que copiamos antes y que abrimos la carpeta anteriormente, entramos en la carpeta /FTBLite2/
  • Borramos las carpetas:
  • /config/
  • /mods/
  • Y copiamos todo lo que hay en el archivo minecraft.zip en la carpeta (Donde antes estaba /mods/, copiamos /mods/, donde antes estaba /config/, copiamos /config/)

Ahora podemos usar el pack de mods del servidor iniciando el juego seleccionando el pack de mods FTB Lite 2.
