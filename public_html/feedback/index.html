About feedback Before reporting a bug, asking for a fix, or giving an idea, first and foremost, mention the game you are referring to. That helps a lot.


How to report a bug To report a bug, first mention the game where you found it, where exactly and the number of steps you followed to find it.

e.g. "I was playing Lemontastic and found a bug in the supermarket. Buying a grapefruit gave me NaN money!"


About fixes Is the game unbalanced? Do you think some skill is too underpowered? A price is too high? This feature used to be cool but needs a revamp? Tell us what you think and we can fix it to balance all games!

e.g. "I think in Lemontastic the Lawyer Raccoon Skill is underpowered. It needs a higher chance!"


About ideas You can submit all ideas you want, but keep in mind that some new features are harder to implement than others.

e.g. "I want a button in Lemontastic located below the Money Bar that easily gives me TONS OF MONEY. Please. The game is too hard without that button!"
